## ❓ Частые вопросы

**Q: Как создать заказ?**  
A: Вкладка "Создать заказ" → выберите клиента и пункт выдачи → добавьте товары → нажмите "Создать заказ"

**Q: Где посмотреть API документацию?**  
A: https://localhost:7147/swagger или файл `REST_API_DOCUMENTATION.md`

**Q: Как тестировать API?**  
A: Используйте Swagger UI или примеры в файле `API_EXAMPLES.md` (cURL, Postman, PowerShell)

**Q: Что делать если не подключается к БД?**  
A: 1) Проверьте MySQL на 192.168.100.35:3306 2) Убедитесь что БД создана 3) Смотрите `DBFORGE_IMPORT_GUIDE.md`

**Q: Приложение не запускается?**  
A: Выполните `dotnet clean` → `dotnet restore` → `dotnet build` → `dotnet run`

## 📊 Пример работы

### Создание заказа через web-интерфейс

1. Нажмите вкладку "Создать заказ"
2. Выберите клиента: "Иван Петров"
3. Выберите пункт выдачи: "Пункт выдачи #1"
4. В первой строке товаров:
   - Выберите: "Ноутбук Dell"
   - Количество: 1
5. Нажмите "+ Добавить товар"
6. Во второй строке:
   - Выберите: "Клавиатура механическая"
   - Количество: 2
7. Нажмите "Создать заказ"
8. ✅ Заказ создан! Смотрите его на вкладке "Заказы"

### Создание заказа через API (cURL)

```bash
curl -X POST "https://localhost:7147/api/orders" \
  -H "Content-Type: application/json" \
  -k \
  -d '{
    "userId": 1,
    "pickupPointId": 1,
    "items": [
      {"productId": 1, "quantity": 1},
      {"productId": 2, "quantity": 2}
    ]
  }'
```
